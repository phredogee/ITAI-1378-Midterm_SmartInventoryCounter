# PPE Safety Detection System

## Project Description

This project develops a computer vision system capable of detecting personal protective equipment (PPE) in workplace environments using YOLO11 object detection.

The model detects:

- Boots
- Gloves
- Helmets
- Human Workers
- Safety Vests

The goal is to automate workplace safety monitoring by identifying whether required PPE is present in images.

## Dataset

Construction PPE Detection Dataset

Classes:
- boots
- gloves
- helmet
- human
- vest

Training Images: 4,380
Validation Images: 420

## Technology Stack

- Python
- YOLO11
- Ultralytics
- PyTorch
- OpenCV
